#  Rock Paper Scissors

A fun and interactive Rock Paper Scissors game built with HTML, CSS, 
and JavaScript where you play against the computer!

---


##  Features

-  **Player vs Computer** — Play against a smart random AI
-  **Winner Detection** — Instantly know who won each round
-  **Score Tracking** — Keeps track of Player & Computer scores
-  **Reset Button** — Restart the game anytime
-  **Animated UI** — Smooth hover & shake animations
-  Fully **Responsive** for mobile and desktop

---

##  How to Play

1. Choose either **Rock** , **Paper** , or **Scissors** 
2. The **computer randomly picks** its choice
3. The result is displayed instantly — **Win**, **Lose**, or **Draw**
4. Scores update automatically after each round
5. Click **Reset** to start fresh anytime

---

##  Game Rules

| Player  | Computer | Result |
|---------|----------|--------|
|  Rock  |  Scissors |  Player Wins |
|  Rock  |  Paper    |  Player Loses |
|  Paper |  Rock     |  Player Wins |
|  Paper |  Scissors |  Player Loses |
|  Scissors |  Paper |  Player Wins |
|  Scissors |  Rock  |  Player Loses |
| Any     | Same     |  Draw |
